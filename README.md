# 复合材料性能 辅助计算程序

> 求解层合板的柔度矩阵、刚度矩阵

项目地址：[GitHub](https://github.com/laorange/LaminateCalculation)  |  [Gitee](https://gitee.com/laorange/LaminateCalculation)

## 快速开始

```mermaid
graph LR
	开始(开始)
	结束(结束)
	第一次使用{第一次使用?}
	
	开始 --> 第一次使用
	第一次使用 --Yes--> 安装python --> 安装依赖库 --> 运行后台
	第一次使用 --No--> 运行后台
	运行后台 --> 按后台提示访问指定网址 --> 输入数据 --> 开始计算 --> 结束
```
